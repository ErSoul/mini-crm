# MINI-CRM

## Installation

```bash
composer install
npm install
npm run dev

php artisan storage:link
php artisan migrate --seed
```

## Use

`php artisan serve`

Login with:

#### Admin (Can create, edit or delete companies and employees)
user: admin@admin.com
pass: password

#### Other (Can view companies or employees)
user: anonymous@example.org
pass: example
